# 贡献指南

十年的 PR 难道不提了吗？提！🖤🎧

## 我们特别欢迎的贡献

### 📖 贡献嘉豪语录
在百科里加入你发现的嘉豪语录、名场面、或嘉豪宇宙新分支：
1. Fork → 编辑 `references/jiahao-encyclopedia.md`
2. 注明出处（哪个平台、什么场景）
3. 按现有格式添加到对应章节
4. 提 PR，标题加 `[lore]` 前缀

### 🐛 报 Bug
用 Bug Report 模板提交 Issue。描述复现步骤和期望行为。

### 💡 提功能建议
用 Feature Request 模板提交 Issue。

### 🔧 改进 Skill 本身
1. Fork → 创建分支 `feat/your-feature`
2. 改 SKILL.md 或相关文件
3. 确保不破坏三层浓度机制
4. 提 PR

## PR 规范
- 描述清楚改了什么、为什么
- 嘉豪语录类贡献附出处
- 代码类贡献确保跨平台兼容

## 开发环境
纯文本 skill，不需要编译。你需要：
- 文本编辑器
- Claude Code 或 Claude.ai 测试
- Python 3（仅脚本需要）
